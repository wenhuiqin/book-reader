## 少数派

标题建议：

`我做了一个更安静的桌面阅读器：Book Reader`

导语：

我一直想找一个更专注于“阅读本身”的桌面应用，而不是功能复杂、信息密度过高的文档管理工具。所以做了 `Book Reader`：一个本地优先、极简、支持 EPUB 与 PDF 的桌面阅读器。

可展开正文：

这个项目目前最核心的思路是三件事：

1. 书架要足够安静，导入本地书就能开始读。
2. 阅读窗口要独立，减少“主窗口 + 面板 + 管理器”对阅读的干扰。
3. 阅读进度和偏好要保存在本地，下一次打开能接着看。

当前版本已经支持：

- 本地导入 EPUB / PDF
- EPUB 元数据、目录、封面与章节阅读
- PDF 基础分页阅读
- 自动滚动与速度调节
- 进度恢复

## Reddit

`I built an open-source local-first desktop reader for EPUB and PDF`

Post body:

I have been experimenting with a simpler reading experience for desktop and ended up building `Book Reader`.

It is an open-source Electron + React app focused on:

- local EPUB and PDF import
- a calm bookshelf UI
- an independent floating reader window
- local reading progress restore
- EPUB metadata, cover, TOC, and chapter parsing
- fine-grained auto-scroll control
